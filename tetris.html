Tetris

Ir al sitio del proyecto

Elaborado en 2017, es un clon del clásico juego.

Fue desarrollado utilizando HTML, CSS y JavaScript, utilizando la API de canvas para elaborar el tablero de juego.

El juego cuenta con varios niveles y registra los puntos ganados por las filas eliminadas.

En 2019 fue mejorado el diseño del juego, entre otros cambios internos.
